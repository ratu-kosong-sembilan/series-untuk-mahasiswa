# Animation QC Checklist - CORE-001

Definition of Ready
- Script lines frozen for the scene
- Timing target agreed (from shot list)
- Visual references provided (colors, axes, labels)

Definition of Done
- Review MP4 approved
- Alpha render delivered (PNG sequence or ProRes 4444)
- Source file delivered (.blend/.aep)
- Naming and folders correct

QC checks
- Labels legible at 1080p
- Colors consistent with spec (Real blue, Imag orange, Magnitude green, Angle purple)
- Motion smooth, no jitter
- Text spelling correct
- Alignment clean and consistent
- Background clean, no artifacts
- Frame rate 30 fps

Delivery naming
- CORE001_SEG{#}_SCN{ID}_vNN.ext
- CORE001_SEG{#}_SCN{ID}_alpha_vNN.mov
