# Naming Conventions — CORE-001 Production

## Files
- Segments: `CORE001_SEG{1-6}_vNN.ext`
- Scenes: `CORE001_SEG{1-6}_SCN{ID}_vNN.ext`
- Audio: `CORE001_SEG{1-6}_VO_vNN.wav`
- Animation: `CORE001_SEG{1-6}_SCN{ID}_alpha_vNN.mov`
- Exports: `CORE001_SEG{1-6}_final_vNN.mp4`

## Versions
- v01 → first draft
- vNN increments per revision

## Folders
- `production/core-001/script` — final voiceover text & cues
- `production/core-001/storyboard` — shot list & animator materials
- `production/core-001/audio` — VO raw and processed
- `production/core-001/video` — editor project and renders
- `production/core-001/animation` — animation sources & outputs
- `production/core-001/assets` — slides, images, logos
- `production/core-001/exports` — review exports
- `production/core-001/delivery` — final assets & metadata
