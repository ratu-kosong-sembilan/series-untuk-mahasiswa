# VO Recording Checklist - CORE-001

## Setup
- Quiet room, AC off, phone silent
- Mic position 15-20 cm, pop filter on
- Sample rate 48 kHz, 24-bit if available
- Peak around -6 dBFS, no clipping

## File naming
- CORE001_SEG1_VO_v01.wav
- CORE001_SEG2_VO_v01.wav
- CORE001_SEG3_VO_v01.wav
- CORE001_SEG4_VO_v01.wav
- CORE001_SEG5_VO_v01.wav
- CORE001_SEG6_VO_v01.wav

## Procedure
1. Record 10 seconds room tone (start and end)
2. Read SEGx_VO.txt slowly, clear diction
3. Pause 1-2 seconds before key equations
4. If mistake, pause and repeat sentence
5. Save raw file to audio/raw/

## QC quick check
- No clipping or distortion
- Consistent volume
- Minimal background noise

## Notes
- Use script files in production/core-001/script/vo/
