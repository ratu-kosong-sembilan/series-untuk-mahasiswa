# Edit QC Checklist - CORE-001

Definition of Ready
- VO WAV exists for the segment
- Required slides/graphics present
- Animations (alpha) present if needed

Definition of Done
- Review MP4 approved
- Loudness near -14 LUFS, peaks below -1 dBTP
- No black frames or stutters
- On-screen text legible
- Subtitles synced

QC checks
- Color and contrast consistent
- Pacing matches VO
- Lower thirds aligned
- No clipping or distortion
- No spelling errors

Delivery naming
- CORE001_SEG{#}_final_vNN.mp4
