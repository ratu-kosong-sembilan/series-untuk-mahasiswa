# Video Editor Brief — CORE-001

## Project Overview
Course: CORE-001 — Bilangan Kompleks untuk Engineering (55 min)
Goal: Assemble voiceover, slides, animations, and simulator captures into clear, paced lessons.

## Inputs
- Voiceover WAV files per segment (48 kHz, 24-bit)
- Slides (PNG/PSD) and on-screen text guidelines
- Animations (MP4 review + alpha renders)
- Screen captures (simulator walkthrough)
- Script cues from [01-MASTER-PLAN/05-CORE-001-PRODUCTION-PLAN.md](../../01-MASTER-PLAN/05-CORE-001-PRODUCTION-PLAN.md)
- Shot list: [production/core-001/storyboard/shot-list-core-001.csv](../storyboard/shot-list-core-001.csv)

## Deliverables
- 6 segment videos (PRE, FEN, VIS, MKN, MOD, SIM)
- Final render: 1080p, 30fps, H.264, -10 to -14 LUFS integrated
- Project file (Premiere/Resolve) with neat bins and labels
- Simple captions (key terms, equations) per script
- Thumbnail suggestions (3 options)

## Editing Guidelines
- Pacing: steady, no rushed cuts; allow absorption time
- Transitions: simple dissolves or motion cuts; avoid heavy effects
- Text: large, high-contrast; avoid over-crowding
- Audio: VO first; music minimal and low (optional)
- Equation rendering: crisp; use vector when possible

## Technical Specs
- Resolution: 1920×1080 (16:9)
- Frame rate: 30 fps
- Audio loudness: -14 LUFS target (YouTube safe), peaks < -1 dBTP
- Color space: Rec. 709
- File naming: CORE001_SEG{1-6}_final_vNN.mp4

- Kickoff: Week 2
- First cuts: Week 3–4
- Finals: Week 5
