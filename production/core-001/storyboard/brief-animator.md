# Animator Brief — CORE-001 (Phase 1 Animations)

## Project Overview
Course: CORE-001 — Bilangan Kompleks untuk Engineering (55 min)
Goal: Deliver clear, intuitive animations for key concepts (phasor rotation, vector projection, polar→complex conversion, parameter sweep results).

## Deliverables (Phase 1)
- 12 short animations (10–45s each), 1080p, 30fps, sRGB
- Transparent-background versions for compositing (ProRes 4444 or PNG sequence)
- Final MP4 H.264 exports for review
- Source project files (AE/Blender) with organized hierarchy
- Simple thumbnail preview images (first frame)

## Animation List (Initial)
1. Phasor rotation overview — smooth circular motion + angle marker
2. Vector projection demo — real/imag axes + dynamic projection
3. Polar→Complex conversion — radius/angle to x+jy transition
4. Parameter sweep result — multi-curve evolution over time
5–12. Additional micro-animations (to be finalized via shot list)

Refer to: [01-MASTER-PLAN/07-ANIMATION-STORYBOARD.md](../../01-MASTER-PLAN/07-ANIMATION-STORYBOARD.md) and the shot list CSV.

## Visual Language & Style
- Clean, minimal, engineering-grade visuals (no gimmicks)
- Color coding: real axis (blue), imaginary axis (orange), magnitude (green), angle (purple)
- Labeling: short, legible, no clutter; font like Inter/Roboto
- Grid/axes where relevant; maintain consistent scale across scenes

## Technical Specs
- Resolution: 1920×1080 (16:9)
- Frame rate: 30 fps
- Color space: sRGB (Rec. 709)
- Delivery: MP4 (review), ProRes 4444 / PNG sequence (final alpha)
- File naming: CORE001_SEG{1-6}_SCN{ID}_vNN.ext

## Workflow & Process
1. Receive shot list and storyboard scene notes
2. Produce low-fidelity animatics for timing approval
3. Iterate to final visuals
4. Deliver review MP4s and source files
