# MODUL: PENTINGNYA ALJABAR & KALKULUS UNTUK EM & TRANSIEN

## Ringkasan
Modul ini menjelaskan mengapa aljabar (termasuk bilangan kompleks) dan kalkulus menjadi fondasi untuk memahami elektromagnetik dan transient. Disajikan dengan pendekatan Fenomena-First: fenomena -> visual -> makna -> model matematis -> simulasi -> aplikasi industri.

## Target
Mahasiswa dan calon mahasiswa teknik elektro / bidang terkait.

## Durasi Total
30-45 menit (video + simulasi web).

## Output
- Video utama 30-45 menit.
- Simulasi web interaktif (slider parameter + grafik).

## Tujuan Pembelajaran
1) Menjelaskan kaitan perubahan waktu (d/dt) dengan medan dan transien.
2) Menafsirkan integral sebagai akumulasi efek listrik.
3) Memahami peran aljabar dan bilangan kompleks untuk fase dan magnitude.
4) Menghubungkan rumus dasar R, L, C dengan fenomena nyata.
5) Menggunakan simulasi untuk mengeksplorasi dampak perubahan parameter.

## Struktur Modul
1) Pre-Module (5 menit)
2) Core Content Tahap 1-6 (25-35 menit)
3) Post-Module (10-12 menit)
4) Hands-On Project (Opsional, 10-15 menit)

## Core Content Ringkas (Tahap 1-6)
- Tahap 1: Fenomena fisik (hook real-world: lampu redup, inrush motor)
- Tahap 2: Visualisasi (kurva waktu, slope, area)
- Tahap 3: Makna fisis (turunan = laju perubahan, integral = akumulasi)
- Tahap 4: Model matematis (v = L di/dt, i = C dv/dt, persamaan RL)
- Tahap 5: Simulasi (slider R, L, C, step input)
- Tahap 6: Aplikasi industri (relay setting, switching surge)

## Assessment
- Kuis 5 pertanyaan konsep
- Latihan soal 2-3 butir (interpretasi grafik)
- Mini project opsional: modifikasi parameter simulator dan jelaskan efek

## Quality Checklist
- Hook 30 detik pertama engaging
- Visual perubahan vs waktu jelas
- Setiap rumus punya makna fisis
- Ada simulasi dengan parameter
- Ada aplikasi industri nyata
