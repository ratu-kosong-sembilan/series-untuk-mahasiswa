## Style Visual
- Warna standar: tegangan biru, arus merah, impedansi hijau.
- Font utama: Sora (judul), Inter (body).
- Grafik: clean, grid halus, label jelas.
- Transisi: cut + wipe sederhana, tidak berlebihan.

## Pipeline Produksi
1. Script final + timecode
2. Storyboard + animatic
3. Asset list + simulasi
4. Produksi animasi per segmen
5. VO recording + SRT
6. Editing + QC
7. Publish + metadata
