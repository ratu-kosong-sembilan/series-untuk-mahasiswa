## Panduan VO
- Pace: 120-140 kata/menit.
- Tone: jelas, instruktif, tidak terlalu cepat.
- Audio target: noise floor < -55 dB, peak -3 dB.
- Rekam per segmen (1-2 menit) untuk memudahkan retake.

## Format SRT (Contoh)
```
1
00:00:00,000 --> 00:00:06,000
Kenapa angka imajiner bisa menjelaskan listrik nyata?
```

## Style Subtitle
- Maks 2 baris, 42 karakter per baris.
- Hindari menumpuk rumus panjang.
- Gunakan istilah konsisten: phasor, impedance, power factor.
