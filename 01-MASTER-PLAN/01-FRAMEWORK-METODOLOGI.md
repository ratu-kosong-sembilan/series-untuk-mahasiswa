Memberikan metodologi produksi end-to-end dari ide hingga publish agar hasil konsisten dan dapat diskalakan.

- Setiap tahapan menghasilkan artefak yang dapat ditinjau tim.
- Checklist operasional tersedia.

## Metodologi Produksi (Ide -> Publish)

### Tahap 0: Riset & Tujuan
**Input:** Pain point audiens, kurikulum, target learning outcomes  
**Output:** Problem map, target audience, KPI pembelajaran  
**Quality Gate:** Problem map disetujui

### Tahap 1: Outline & Struktur
**Input:** Problem map, learning objectives  
**Output:** Outline segmen, durasi target, arc pembelajaran  
**Quality Gate:** Outline lolos review dosen/editor

### Tahap 2: Script (VO + On-screen)
**Input:** Outline, referensi teknis  
**Output:** Naskah lengkap, daftar on-screen text, CTA  
**Quality Gate:** Konsistensi istilah dan durasi

### Tahap 3: Storyboard & Animatic
**Input:** Script  
**Output:** Storyboard per scene, animatic rough cut  
**Quality Gate:** Pacing dan visual clarity

### Tahap 4: Asset & Simulasi
**Input:** Storyboard, simulasi spec  
**Output:** Asset list, simulasi working, UI wireframe  
**Quality Gate:** Asset checklist 80% ready

### Tahap 5: Produksi Animasi
**Input:** Animatic, asset final  
**Output:** Render scene per segmen  
**Quality Gate:** QC visual (warna, timing, alignment)

### Tahap 6: Voiceover & Subtitle
**Input:** Script final  
**Output:** VO bersih, SRT sinkron  
**Quality Gate:** Noise < -55 dB, pacing sesuai

### Tahap 7: Editing & Publish
**Input:** Render scene, VO, SRT  
**Output:** Master 16:9, Shorts 9:16, metadata  
**Quality Gate:** QC teknis dan QA konten

## Checklist Operasional
- [ ] Semua file mengikuti naming convention
- [ ] Durasi sesuai target (40:00 long form)
- [ ] VO sinkron dengan animasi
- [ ] SRT tervalidasi di editor
- [ ] QC teknis (fps, resolusi, audio)
- [ ] Publish checklist terpenuhi
